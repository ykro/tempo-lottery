The frontend application for the Tempo Lottery game. Built with Next.js, Tailwind CSS, and Framer Motion.

## Features

- **Visual Play**: Gameplay focused on visual recognition.
- **Dynamic Difficulty**: Timer accelerates as the deck progresses.
- **Mobile Optimized**: Swipe gestures for deck viewing.
- **PWA Ready**: Visual assets and meta tags tuned for installation.

## Development

```bash
npm install
npm run dev
```

## Deployment

Uses `deploy.sh` to deploy to Google Cloud Run via Buildpacks.

```bash
./deploy.sh
```
